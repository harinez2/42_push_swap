# 42_push_swap

## Usage
```
./push_swap [OPTIONS] ARRAY_OF_INT
```

### OPTIONS
-v					Show contents of the stack.
-c					Add color to the operations.
### ARRAY_OF_INT
Int array, each separated by a space.

## Example
When you run the following command:
```
./push_swap 0 2 1
```

Then, You'll get the follwoing results.
```
sa
ra
```
